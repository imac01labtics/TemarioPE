Tema 1: Estadística descriptiva
Subtema Definición
1.1 Conceptos básicos de estadística Definición: La estadística es la disciplina que se encarga de recopilar, organizar, analizar e interpretar datos para la toma de decisiones.
Teoría de decisión: Método estadístico para tomar decisiones considerando la incertidumbre y las probabilidades.
Población: Conjunto completo de elementos o individuos sujetos a estudio.
Muestra aleatoria: Subconjunto representativo de la población, seleccionado de manera que cada individuo tenga igual probabilidad de ser incluido.
Parámetros aleatorios: Valores desconocidos que describen características de una población.
1.2 Descripción de datos Datos agrupados y no agrupados: Los datos no agrupados son aquellos que se presentan individualmente, mientras que los datos agrupados están organizados en categorías o intervalos.
Frecuencia de clase: La frecuencia de clase es el número de veces que se observa un valor dentro de un intervalo particular en un conjunto de datos agrupados.
Frecuencia relativa: La frecuencia relativa es la proporción o porcentaje de veces que ocurre un valor dentro de un conjunto de datos.
Punto medio: El punto medio es el valor medio de un intervalo en datos agrupados.
Límites: Los límites son los valores extremos de un intervalo en datos agrupados.
1.3 Medidas de tendencia central Media aritmética: Es el promedio comúnmente utilizado, obtenido sumando todos los valores y dividiendo esa suma por la cantidad de valores.
Media geométrica: Es el promedio obtenido multiplicando todos los valores y luego tomando la raíz n-ésima del producto, donde n es el número total de valores.
Media ponderada: Es un promedio donde cada valor tiene un peso asignado. Se calcula multiplicando cada valor por su respectivo peso, sumando estos productos y dividiendo por la suma de los pesos.
Mediana: Valor central de un conjunto de datos ordenados.
Moda: Valor más frecuente en un conjunto de datos.
Medidas de dispersión: Indican la variabilidad de los datos respecto a una medida central.
Varianza: Es una medida de dispersión que indica cuánto se alejan los valores de una serie de datos de su media. Se calcula como el promedio de las diferencias al cuadrado entre cada dato y la media.
Desviación estándar: Es la raíz cuadrada de la varianza. Proporciona una medida de dispersión similar a la varianza, pero en la misma escala que los datos originales.
Desviación media: Es la media de las diferencias absolutas entre cada dato y la media. Proporciona una medida de dispersión que considera tanto los valores por encima como por debajo de la media.
Desviación mediana: Es la mediana de las diferencias absolutas entre cada dato y la mediana. Es menos sensible a valores extremos que la desviación media, lo que la hace útil en conjuntos de datos con valores atípicos.
Rango: Es la diferencia entre el valor máximo y el valor mínimo en un conjunto de datos. Es una medida simple de dispersión que puede proporcionar una idea general de la amplitud de los datos.
1.4 Parámetros para datos agrupados Este tema se centra en los métodos para calcular parámetros estadísticos, como la media y la varianza, cuando los datos están agrupados en intervalos en lugar de ser valores individuales. Esto es importante porque muchos conjuntos de datos son demasiado grandes para analizar cada valor individualmente, por lo que se agrupan en intervalos para facilitar su análisis.
1.5 Distribución de frecuencias Se aborda cómo organizar los datos en categorías o intervalos y contar cuántas observaciones caen en cada categoría. La distribución de frecuencias proporciona una forma clara de comprender la distribución de los datos y es fundamental para calcular estadísticas descriptivas y hacer inferencias sobre la población a partir de una muestra.
1.6 Técnicas de agrupación de datos Se exploran los métodos para agrupar datos en intervalos o categorías de manera significativa y útil. Esto puede incluir el uso de reglas específicas para determinar el ancho de los intervalos y garantizar que la agrupación refleje adecuadamente la distribución de los datos.
1.7 Técnicas de muestreo En el desarrollo de este tema se examinan los diferentes métodos para seleccionar una muestra representativa de una población más grande. Estas técnicas son cruciales para garantizar que la muestra captura adecuadamente la variabilidad y características de la población, lo que permite hacer inferencias precisas sobre la población a partir de la muestra seleccionada.
1.8 Histogramas Parecidos a los gráficos de barras, estos muestran la distribución de frecuencias de un conjunto de datos.
Ayudan a visualizar la forma y dispersión de los datos, identificando patrones y tendencias. Son útiles para datos agrupados y continuos.

Tema 2: Fundamentos de la Teoría de Probabilidad
Subtema Definición
2.1 Técnicas de Conteo Las técnicas de conteo son herramientas fundamentales en probabilidad y estadística que nos permiten calcular el número de maneras en las que puede ocurrir un evento o una serie de eventos. Estas técnicas son esenciales para determinar las probabilidades en diferentes situaciones.
2.1.1 Principio aditivo El principio aditivo se utiliza cuando se desea contar el número total de maneras en las que puede ocurrir uno de varios eventos mutuamente excluyentes. Si un evento A puede ocurrir de nA maneras y un evento B puede ocurrir de nB maneras, y los dos eventos no pueden ocurrir simultáneamente, entonces el número total de maneras en las que puede ocurrir A o B es:
nA + nB
2.1.2 Principio Multiplicativo El principio multiplicativo se aplica cuando se desea contar el número total de maneras en las que puede ocurrir una serie de eventos independientes. Si un evento A puede ocurrir de nA maneras y un evento B puede ocurrir de nB maneras, entonces el número total de maneras en las que pueden ocurrir ambos eventos es:
nA ⋅ nB
2.1.3 Notación Factorial La notación factorial se utiliza para contar el número de maneras en las que se pueden ordenar n objetos distintos. El factorial de un número n, denotado como n!, se define como:
n! = n ⋅ (n - 1) ⋅ (n - 2) ... 3 ⋅ 2 ⋅ 1
2.1.4 Permutaciones Las permutaciones se refieren al número de maneras de ordenar un conjunto de objetos. Si se tienen n objetos y se desean ordenar todos, el número de permutaciones es n!. Si se desea ordenar solo k objetos de un conjunto de n, el número de permutaciones se calcula como:
P(n,k) = n! (n-k)!
2.1.5 Combinaciones Las combinaciones se refieren al número de maneras de seleccionar k objetos de un conjunto de n sin importar el orden. El número de combinaciones se calcula como:
C(n,k) = n k = n! k!(n-k)!
2.1.6 Diagrama de Árbol Un diagrama de árbol es una representación gráfica que muestra todas las posibles combinaciones o resultados de una serie de eventos. Se utiliza para visualizar y calcular las probabilidades y posibles resultados en escenarios con múltiples etapas.
2.1.7 Teorema del Binomio El teorema del binomio proporciona una fórmula para expandir potencias de binomios. Para cualquier número real a y b, y un número entero no negativo n.
2.2 Teoría elemental de probabilidad La teoría elemental de probabilidad establece los principios básicos para calcular probabilidades, incluyendo los axiomas de Kolmogorov que definen propiedades esenciales como no negatividad, certeza y aditividad de las probabilidades.
2.3 Probabilidad de Eventos Espacio Muestral y Evento
Espacio muestral: Conjunto de todos los posibles resultados de un experimento.
Evento: Subconjunto del espacio muestral.
Simbología y Operaciones
Unión (A∪B): Evento que ocurre si A o B ocurren.
Intersección (A∩B): Evento que ocurre si A y B ocurren.
Diagramas de Venn: Herramienta gráfica para representar eventos y sus relaciones.
2.4 Probabilidad con Técnicas de Conteo Utiliza combinaciones y permutaciones para contar el número de formas en que pueden ocurrir eventos, aplicando axiomas y teoremas como el de la probabilidad total y el de Bayes para calcular probabilidades.
2.5 Probabilidad Condicional La probabilidad condicional P(A|B) mide la probabilidad de que ocurra el evento A dado que B ha ocurrido. Los eventos pueden ser dependientes (la ocurrencia de uno afecta al otro) o independientes (la ocurrencia de uno no afecta al otro).
2.6 Ley Multiplicativa Calcula la probabilidad de la intersección de dos eventos:
Eventos dependientes:
P(A∩B)=P(A)⋅P(B∣A)
Eventos independientes:
P(A∩B)=P(A)⋅P(B)
2.7 Eventos Independientes: Regla de Bayes La regla de Bayes permite actualizar la probabilidad de un evento basado en nueva información, calculando P(Bi|A) a partir de P(A|Bi) y P(Bi) .

Tema 3: Variables Aleatorias
Subtema Definición
3.1 Variables aleatorias discretas Una variable aleatoria es un medio para describir los resultados del espacio muestral empleando números, es decir, es una función1 que asigna valores numéricos a cada uno de los eventos del espacio muestral. Puede ser discreta o continua y se acostumbra representarla con una X.
3.1.1 Distribución de Probabilidad en Forma General La distribución de probabilidad de una variable aleatoria discreta es una función que asigna a cada posible valor de la variable su probabilidad correspondiente. Formalmente, si X es una variable aleatoria discreta, entonces su distribución de probabilidad se define como:
P(X = xi) = pi
Donde:
xi es un valor posible de X
pi es la probabilidad de que X tome el valor xi
3.1.2 Valor Esperado El valor esperado, o esperanza matemática, de una variable aleatoria discreta es una medida del centro de su distribución de probabilidad. Se define como la suma ponderada de todos los posibles valores de la variable, donde los pesos son las probabilidades de cada valor. Formalmente:
E(X) = ∑ xi pi
Donde:
xi son los valores posibles de X
pi son las probabilidades correspondientes
3.1.3 Varianza y Desviación Estándar La varianza de una variable aleatoria discreta es una medida de la dispersión de sus valores alrededor del valor esperado. Se define como la esperanza del cuadrado de la diferencia entre la variable y su valor esperado:
Var ( X ) = ∑ i ( x i - E ( X ) ) 2 p ( x i )
La desviación estándar es la raíz cuadrada de la varianza y proporciona una medida de dispersión en las mismas unidades que la variable original:
σ X = Var ( X )
3.1.4 Función de Distribución Acumulada (FDA) La función de distribución acumulada (FDA) de una variable aleatoria discreta X es una función que da la probabilidad de que X tome un valor menor o igual a un valor dado x. Se define como:
F X ( x ) = P ( X ≤ x ) = ∑ x i ≤ x P ( X = x i )
3.2 Variables Aleatorias Continuas Una variable aleatoria continua puede tomar cualquier valor dentro de un intervalo de la recta real. Estas variables se describen mediante funciones de densidad de probabilidad en lugar de funciones de probabilidad.
3.2.1 Distribución de Probabilidad en Forma General La distribución de probabilidad de una variable aleatoria continua se describe mediante una función de densidad de probabilidad (fdp), denotada como fX(x) .
Las propiedades principales de la fdp son:
fX(x) ≥ 0 para todo x.
La integral de fX(x) sobre todo el espacio es igual a 1:
∫ -∞ ∞ f X ( x ) d x = 1
3.2.2 Valor Esperado El valor esperado (esperanza matemática) de una variable aleatoria continua X se calcula como:
E ( X ) = ∫ -∞ ∞ x f X ( x ) d x
3.2.3 Varianza y Desviación Estándar La varianza de una variable aleatoria continua X se define como:
Var ( X ) = ∫ -∞ ∞ ( x - E ( X ) ) 2 f X ( x ) d x
La desviación estándar es la raíz cuadrada de la varianza:
σ X = Var ( X )
3.2.4 Función de Distribución Acumulada (FDA) La función de distribución acumulada (FDA) de una variable aleatoria continua X se define como:
F X ( x ) = P ( X ≤ x ) = ∫ -∞ x f X ( t ) d t
3.2.5 Cálculos de Probabilidad Para una variable aleatoria continua X, la probabilidad de que X tome un valor dentro de un intervalo [a,b] se calcula como la integral de la función de densidad de probabilidad sobre ese intervalo:
P ( a ≤ X ≤ b ) = ∫ a b f X ( x ) d x
